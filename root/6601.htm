The reason why standard telephone modems are so slow is because before each bit is sent, the modem sends 'Are you sure you would like to receive this bit?' and waits for a reply of 'I certainly am sure.' before sending. If not for the poor planning of modem standards, telephone modems would be able to transmit data at incredible rates that would rival T3 speeds.
Brian Scholer
